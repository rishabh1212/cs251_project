Box2D : Project Car

This a project based on the Rubegoldberg Machine with the physics engine by Box2D.

We were inspired from https://code.google.com/p/box2d-html5/ to do this project

It majorly involves a car and different terrain for the car to move using different simulations in. There are a lot of keys involved to move the car and rotate it and stop it.
